# Community Hero: Civic Issue Tracker

Community Hero ek comprehensive **Civic Issue Reporting Platform** hai jahan citizens apne area ki problems (jaise road damage, water leakage, etc.) report kar sakte hain aur local authorities unhe track aur resolve karti hain.

## 🚀 Features
* **User Authentication:** Secure JWT-based Login/Signup for Citizens and Admins.
* **Smart Reporting:** Citizens apni problems report kar sakte hain, location aur media (image/video) upload ke saath.
* **AI Integration:** Google Gemini API ka use karke issues ka auto-categorization aur urgency analysis.
* **Real-time Notifications:** Status update hone par automatic notification system.
* **Admin Dashboard:** Admins issues ko track, update (status change), aur manage kar sakte hain.
* **Security:** Role-based access control (RBAC).

## 🛠 Tech Stack
* **Frontend:** React.js, Tailwind CSS, Lucide-react (Icons).
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB (Mongoose).
* **AI/API:** Google Gemini API (Analysis), Nominatim (Geocoding), Cloudinary (Media storage).
